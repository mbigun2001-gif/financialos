# Вирішення проблем з підключенням до localhost

## Проблема: Safari не може підключитися до localhost:3015

### Рішення 1: Перевірте, чи запущений сервер

1. Відкрийте термінал і перейдіть до папки проекту:
```bash
cd /Users/mbigun/Desktop/financial-os
```

2. Запустіть сервер:
```bash
npm run dev
```

3. Чекайте повідомлення:
```
✓ Ready in X seconds
○ Local: http://localhost:3015
```

### Рішення 2: Перевірте правильну адресу

У Safari відкрийте:
- `http://localhost:3015` (не `https://`)
- Або `http://127.0.0.1:3015`

### Рішення 3: Перевірте налаштування Safari

1. Відкрийте **Safari → Налаштування → Конфіденційність**
2. Переконайтеся, що **"Запобігання міжсайтовому відстеженню"** не блокує localhost
3. Спробуйте в режимі приватного перегляду

### Рішення 4: Очистіть кеш та перезапустіть

```bash
# Зупиніть сервер (Ctrl+C)
# Очистіть кеш Next.js
rm -rf .next
# Перезапустіть
npm run dev
```

### Рішення 5: Перевірте, чи порт не зайнятий

```bash
lsof -ti:3015
```

Якщо виводить число - порт зайнятий. Зупиніть процес:
```bash
kill -9 $(lsof -ti:3015)
```

### Рішення 6: Використайте інший браузер для тестування

Спробуйте Chrome або Firefox, щоб перевірити, чи проблема саме в Safari.

### Рішення 7: Перевірте файрвол

1. Відкрийте **System Settings → Network → Firewall**
2. Переконайтеся, що Node.js дозволено приймати вхідні з'єднання

### Якщо нічого не допомагає:

1. Перезапустіть комп'ютер
2. Перевірте, чи немає помилок у терміналі при запуску `npm run dev`
3. Спробуйте запустити на іншому порті:
   - Змініть в `package.json`: `"dev": "next dev -p 3000"`
   - Відкрийте `http://localhost:3000`
