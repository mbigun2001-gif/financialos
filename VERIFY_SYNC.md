# Перевірка роботи синхронізації Supabase

## Як перевірити що синхронізація працює:

1. **Відкрийте консоль браузера** (F12 → Console)

2. **Перевірте підключення до Supabase:**
   - Має з'явитися повідомлення "Дані успішно збережено в Supabase" або "Дані успішно завантажено з Supabase"
   - Якщо бачите помилки - перевірте чи правильно налаштовані змінні оточення

3. **Перевірте в Supabase Dashboard:**
   - Відкрийте Supabase Dashboard
   - Перейдіть в **Table Editor** → **sync_data**
   - Має з'явитися запис з вашим `user_id` та даними

4. **Тестування синхронізації між пристроями:**
   - Відкрийте сайт на телефоні та ноутбуці
   - Додайте транзакцію на одному пристрої
   - Зачекайте 10 секунд
   - Перевірте чи з'явилася транзакція на іншому пристрої

## Якщо синхронізація не працює:

1. Перевірте `.env.local` файл:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=https://jmhvlboizpvdxtxqovhe.supabase.co
   NEXT_PUBLIC_SUPABASE_ANON_KEY=ваш_ключ
   ```

2. Перевірте чи таблиця `sync_data` створена в Supabase

3. Перевірте консоль браузера на помилки

4. Перезапустіть dev сервер після зміни `.env.local`
